Pluit Carousel – Prototype Carousel

by Herryanto Siatono

Description

One of the most widely used Javascript widgets is a carousel, it is used all over the places from photo slides, feature highlights, page browsing, top stories panel, and others. Same concept but different looks and feels.

Pluit Carousel is a simple carousel written to be as bare and simple as possible so you can easily customize to meet your carousel needs. The idea is to use the same semantically structured HTML and Javascript initialization codes, and applying different skins for customized look and feel. If you need more advanced behavior, you can easily extend Pluit.Carousel.

Usage

  1. Include the CSS: 
      <link rel="stylesheet" href="css/pluit-carousel.css" type="text/css" media="screen">
  2. Create the HTML tags: 
      <div class="pluit-carousel">
    <div class="viewport">
      <ul>
        <li>
          <img src="http://l.yimg.com/g/images/home_photo_megansoh.jpg" alt="Photo by megansoh" width="500" height="333" />
        </li>
        <li>
          <img src="http://l.yimg.com/g/images/home_photo_ccgd.jpg" alt="Photo by ccgd" width="500" height="333" />
        </li>
        <li>
          <img src="http://l.yimg.com/g/images/home_photo_lyn.jpg" alt="Photo by lyn" width="500" height="333" />
        </li>
      </ul>
    </div>
  </div>
  3. Include and initialize the JavaScripts: 
      <script type="text/javascript" src="js/prototype.js"></script>
  <script type="text/javascript" src="js/scriptaculous.js"></script>
  <script type="text/javascript" src="js/effects.js"></script>
  <script type="text/javascript" src="js/pluit-carousel.js"></script>
  <script type="text/javascript">
    new Pluit.Carousel.init({
      circular: true
    });
  </script>
  4. Customize your carousel style, sample customized skins are included in css/pluit-carousel-skins.css, append to pluit-carousel class: 
    5.   <div class="pluit-carousel custom-skin">
  ...
  </div>

Options

Name Default
circular false
hideAutoNav false
animDuration 0.5
viewportClassName viewport
navClassName nav
prevClassName prev
nextClassName next
pagesClassName pages
pageClassNamePrefix page-
activeClassName active
